# 홈(/) 페이지 운영 문서

## 1. 문서 정보
- 라우트: `/`
- 페이지 파일: `src/app/page.tsx`
- 문서 파일: `docs/pages/home.page.md`

## 2. 목적
- 홈 루트 진입 시 실제 랜딩 페이지인 `/about`으로 즉시 이동시킨다.
- 루트 경로를 별도 화면으로 유지하지 않고, 단일 진입점을 고정한다.

## 3. 화면/기능 구성
- 렌더링 UI 없음
- 서버 컴포넌트에서 `redirect('/about')`를 호출해 즉시 리다이렉트

## 4. 데이터/상태
- 외부 API/상태 없음
- 클라이언트 상태 없음
- 핵심 의존성: `next/navigation`의 `redirect`

## 5. 인터랙션 규칙
- 사용자가 `/`에 접근하면 브라우저에 `/about`이 표시되어야 한다.
- 중간 로딩 UI를 별도로 노출하지 않는다.

## 6. QA 체크리스트
- [ ] `/` 직접 접속 시 `/about`으로 이동한다.
- [ ] 새로고침 후에도 동일하게 `/about`으로 이동한다.
- [ ] 루트 페이지에 불필요한 본문 UI가 노출되지 않는다.
