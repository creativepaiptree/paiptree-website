# /news 페이지 운영 문서

## 1. 문서 정보
- 라우트: `/news`
- 페이지 파일: `src/app/news/page.tsx`
- 문서 파일: `docs/pages/news.page.md`

## 2. 목적
- 기존 뉴스 경로를 신규 뉴스룸 경로(`/newsroom`)로 통합한다.
- 외부 링크/북마크의 하위 호환성을 유지한다.

## 3. 화면/기능 구성
- 별도 렌더링 UI 없음
- 서버 컴포넌트에서 `redirect('/newsroom')` 수행

## 4. 데이터/상태
- 외부 데이터 없음
- 로컬 상태 없음

## 5. 인터랙션 규칙
- `/news` 접근 시 즉시 `/newsroom`으로 이동
- 중간 안내 메시지나 로딩 화면을 노출하지 않음
