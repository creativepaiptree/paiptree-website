## 핵심 반영 사항
- `체중분포` 헤더 영역의 `N=`, `목표범위`, `전일 비교` 배치가 좁은 화면에서 밀림/잘림 없이 보이도록 레이아웃을 재정렬했습니다.
- `CCTV 무게예측`의 D-1/D-2/D-3 정확도 카드가 모바일에서 가로 한 줄로 과밀 배치되는 문제를 세로 스택 방식으로 바꿨습니다.
- 모바일/태블릿 전환 시 글자 크기, 입력 박스 너비, 버튼 폭을 단계적으로 줄여 터치성 및 가독성을 개선했습니다.
- `정확도 보기` 버튼을 모바일에서 전체 폭(`w-full`)으로 변경해 작은 화면에서 오동작 포인트를 줄였습니다.

## 사용자 피드백 반영 포인트
- 아이폰14 맥스(430x932) 기준 텍스트 밀림 및 잘림 이슈 대응
- "가로값 작아지면 깨짐" 우려를 방지하기 위해 `wrap`/`column` 우선 설계 적용
- `D-1, D-2, D-3`은 모바일에서 가로 라인에 강제 묶지 않고 세로 분리 표시하도록 정책 변경
- "폰트가 작아져야 할 부분"은 우선 `w-full`/`flex-direction` 수정 후 보조적으로 축소 조정
